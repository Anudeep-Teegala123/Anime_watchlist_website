# 🌌 OtakuVault - Anime Watchlist Website

Welcome to **OtakuVault**, a premium, interactive, and beautifully designed web application for tracking your anime journey.

🔗 **Live Demo**: [https://anudeep-teegala123.github.io/Anime_watchlist_website/](https://anudeep-teegala123.github.io/Anime_watchlist_website/)

## ✨ Features
* **Modern Aesthetic**: Glassmorphism design with a dark theme, neon gradients, and smooth interactive animations.
* **Live Analytics Dashboard**: Track total entries, active shows, completed shows, planning list, and your average score in real-time.
* **Instant Filtering & Sorting**: Search by title, filter by watch status, and sort by title, rating, or progress.
* **Quick Progress Tracker**: Increment your watched episodes directly from the list with a single click.
* **Custom Card Themes**: Assign vibrant gradient themes (Cyberpunk, Neon Ocean, Sunset, Emerald) or custom image URLs to your entries.
* **Persistent Storage**: Saves your list automatically using the browser's `localStorage`.

## 🛠️ Technology Stack
* **Markup**: HTML5 (Semantic elements)
* **Styling**: Vanilla CSS3 (Custom properties, grid, flexbox, transitions, backdrop-filter)
* **Behavior**: JavaScript (ES6+, LocalStorage, dynamic DOM manipulation)

## 🚀 Getting Started
Simply open `index.html` in your favorite web browser, or launch a local live server.
